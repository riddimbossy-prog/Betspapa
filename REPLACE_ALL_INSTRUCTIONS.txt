BETSPAPA FULL REPOSITORY REPLACEMENT — v1.4.0
================================================

1. Extract this ZIP.
2. Copy every file and folder into your local Betspapa GitHub repository.
3. Choose Replace/Overwrite All.
4. Commit: BetsPapa v1.4 modern fonts and live dashboard
5. Click Push origin.
6. Wait for both GitHub Pages and Render to finish deploying.
7. Open once:
   https://betspapa.com/reset.html
8. Then open:
   https://api.betspapa.com/api/health
   The version should be 1.4.0 and database should say connected.
9. Open:
   https://api.betspapa.com/api/dashboard/today
   This should return predictions, fixtures, recentResults and stats.
10. Refresh betspapa.com.

The site no longer uses demonstration data. When today has no qualified predictions,
it displays imported fixtures and a clear “No qualified pick yet” state.

Do not commit API keys, Supabase service-role keys or ADMIN_SYNC_SECRET to GitHub.
