BETSPAPA FULL REPOSITORY REPLACEMENT — v1.3.0
================================================

1. Back up your current local Betspapa repository.
2. Extract this ZIP.
3. Copy every file and folder into your local Betspapa repository.
4. Choose Replace/Overwrite All when Windows asks.
5. Open GitHub Desktop.
6. Commit with: BetsPapa v1.3 official Papa branding
7. Click Push origin.
8. GitHub Pages will redeploy the website.
9. Render will redeploy the backend because the server folder is included.
10. On phone/desktop, hard refresh the site or reopen it after the new service-worker cache activates.

This ZIP is complete. It includes the frontend, responsive mobile/tablet/Z Fold UI,
official logo assets, Render backend, Supabase integration, prediction engine,
PowerShell scripts, workflows, tests and documentation.

Do not add API keys or Supabase service-role secrets to GitHub.
