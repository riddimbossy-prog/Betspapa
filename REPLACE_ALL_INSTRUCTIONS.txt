   https://betspapa.com/reset.html

   https://betspapa.com/?tour=1

PWA ICON NOTE
-------------
Android and Samsung Internet may keep an old installed icon until the existing
app is uninstalled and installed again. The website itself uses fresh asset
filenames, so a new installation receives the official branded icon.

No Supabase migration and no new Render environment variable are required.
