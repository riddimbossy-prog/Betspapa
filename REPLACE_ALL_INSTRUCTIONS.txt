BETSPAPA FULL REPOSITORY REPLACEMENT — v1.17.1
================================================

PURPOSE
-------
Engine-only correction for the uploaded v1.17.0 Consensus Banker build.
The frontend, Consensus Banker page, Boss Picks, PWA and settlement workflows
are preserved.

WHAT CHANGED
------------
- Papa's Pick now uses the audited full-market overhaul as its authoritative core.
- Every market is scored independently with its own threshold and blockers.
- The v1.17 common-sense, venue and four-engine consensus layers remain active.
- Duplicate markets inherit later common-sense blockers.
- Explicit odds-aware upgrades remain active.
- Engine version changed to papasense-v1.17.1-overhaul.

DEPLOYMENT
----------
1. Extract this ZIP.
2. Copy every file and folder into your local Betspapa repository.
3. Choose Replace/Overwrite All.
4. Commit:
   BetsPapa v1.17.1 full market engine overhaul
5. Push origin.
6. Wait for Render and GitHub Actions to finish.
7. Open:
   https://api.betspapa.com/api/health
8. Confirm:
   version: 1.17.1
   engineVersion: papasense-v1.17.1-overhaul
   database: connected
9. Run the BetsPapa Automatic Picks workflow once so current dates are regenerated
   under the new engine version.

No Supabase migration or new environment variable is required.
Never commit API keys, Supabase service-role keys or ADMIN_SYNC_SECRET.
