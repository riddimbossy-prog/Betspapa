BETSPAPA FULL REPOSITORY REPLACEMENT — v1.10.2
================================================

FIXED
-----
The prediction cards are HTML buttons. Some browsers were applying black
default button text, making the teams and selections almost invisible on the
dark-purple cards.

v1.10.2 explicitly restores:
- white team names
- white prediction selections
- readable fixture details
- green market labels
- gold confidence figures
- visible keyboard focus

DEPLOY
------
1. Extract this ZIP.
2. Replace every file in the Betspapa repository.
3. Commit:
   BetsPapa v1.10.2 match card contrast fix
4. Push origin.
5. Wait for GitHub Pages and Render.
6. Confirm:
   https://api.betspapa.com/api/health
   Version must show 1.10.2.
7. Open:
   https://betspapa.com/reset.html

No workflow run or prediction regeneration is required because the engine logic
did not change.
