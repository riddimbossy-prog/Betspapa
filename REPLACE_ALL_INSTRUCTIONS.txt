DIAGNOSTICS LOGIN
-----------------
Use the same ADMIN_SYNC_SECRET already stored in Render.
The page stores it only for the current browser tab.

BANKER RULE
-----------
No match is forced into the banker board. Each engine can publish zero to three
strict selections. A pick must be Qualified, have adequate confidence, pass
individual sample requirements and contain no critical caution.

ANTI-ZOMBIE RULE
----------------
Three or more fixtures sharing the same evidence fingerprint and four-engine
score pattern are withheld automatically for review.

Do not commit API keys, Supabase service-role keys or ADMIN_SYNC_SECRET.
