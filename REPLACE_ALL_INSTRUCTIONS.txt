BETSPAPA FULL REPOSITORY REPLACEMENT — v1.10.1
================================================

DEPLOY
------
1. Extract this ZIP.
2. Replace every file in your local Betspapa repository.
3. Commit:
   BetsPapa v1.10.1 readability admin mobile nav
4. Push origin.
5. Wait for GitHub Pages and Render to deploy.
6. Confirm:
   https://api.betspapa.com/api/health
   Version must show 1.10.1.
7. Open once:
   https://betspapa.com/reset.html

PRIVATE ADMIN URL
-----------------
Diagnostics has been removed from the public navigation.

Use:
https://betspapa.com/admin/

The admin page still requires the same ADMIN_SYNC_SECRET stored in Render.
The separate URL and robots exclusion improve separation, but the secret is
the actual security control.
