BETSPAPA FULL REPOSITORY REPLACEMENT — v1.6.0
================================================

1. Extract this ZIP.
2. Copy every file and folder into your local Betspapa GitHub repository.
3. Choose Replace/Overwrite All.
4. Commit:
   BetsPapa v1.6 full market intelligence overhaul
5. Push origin.
6. Wait for GitHub Pages and Render deployments.
7. Open once:
   https://betspapa.com/reset.html
8. Confirm:
   https://api.betspapa.com/api/health
   Version must show 1.6.0 and engineVersion must show papasense-v1.6 when returned.
9. Open betspapa.com.

Because the engine version changed, the first dashboard request automatically
regenerates the selected date with PapaSense v1.6. Allow the first load several
seconds when many fixtures exist.

Every fixture receives one direction. Qualified picks passed their market-specific
checks. Directional picks are the best available reading but are not bankers.

Do not commit API keys, Supabase service-role keys or ADMIN_SYNC_SECRET.
