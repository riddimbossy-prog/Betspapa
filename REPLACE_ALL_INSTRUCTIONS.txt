BETSPAPA FULL REPLACEMENT v1.3.1
================================

1. Extract this ZIP.
2. Copy every file and folder into your local Betspapa GitHub repository.
3. Choose Replace/Overwrite All.
4. Commit: BetsPapa v1.3.1 search cache hard fix
5. Push origin.
6. Wait for GitHub Pages and Render deployments to finish.
7. Open this URL once:
   https://betspapa.com/reset.html
8. It will clear the old service worker/cache and return you to the homepage.

The search box will now remain closed until the search icon is clicked.

Do not put private API keys, Supabase service-role keys or ADMIN_SYNC_SECRET in GitHub.
