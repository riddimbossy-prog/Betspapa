BETSPAPA FULL REPOSITORY REPLACEMENT — v1.16.1

THIS BUILD PRESERVES
--------------------
- Every v1.16 branded PWA and first-user walkthrough feature.

FIXED IN v1.16.1

- API timeout reduced from 35–45 seconds to 9 seconds.

- Temporary API failures no longer erase picks already on screen.
- Dashboard and Results endpoints no longer wait for live-score refresh work.
- Results requests are split into safe Supabase batches.
- Historical published prediction versions are included in Results.
- FT, AET, PEN, AWD and WO finished states are supported.
- Results has saved-data fallback, clear errors and a working retry button.
- New service-worker cache forces the repaired JavaScript to load.
