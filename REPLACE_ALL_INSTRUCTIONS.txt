BETSPAPA FULL REPLACEMENT v1.17.3
=================================

1. Extract this ZIP.
2. Replace every file and folder in the local BetsPapa repository.
3. Commit:
   BetsPapa v1.17.3 day-ahead board preparation
4. Push origin.
5. Wait for GitHub Pages and Render deployments.
6. Confirm:
   https://api.betspapa.com/api/health
   Version must show 1.17.3.
7. In GitHub Actions, run:
   BetsPapa Prepare Tomorrow Board
8. Leave date blank to prepare tomorrow automatically.
9. Confirm tomorrow's status:
   https://api.betspapa.com/api/board-preparation/status?date=YYYY-MM-DD

The workflow automatically runs at 17:25 UTC and 22:25 UTC daily.
Tomorrow's Papa's Pick, Aggressive, Safer, Venue Pattern and consensus data
will be stored before match day.

Do not commit ADMIN_SYNC_SECRET or API keys.
