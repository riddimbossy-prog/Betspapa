2. Copy every file and folder into your local Betspapa GitHub repository.
3. Choose Replace/Overwrite All.
4. Commit:
   BetsPapa v1.18.1 No Draw misfire guard
5. Push origin.
6. Wait for GitHub Pages and Render to finish deploying.
7. Confirm:
   https://api.betspapa.com/api/health

Expected:
   version: 1.18.1
   engineVersion: papasense-v1.18.1-no-draw-guard
   database: connected

8. Clear the previous PWA cache once:
   https://betspapa.com/reset.html

No Supabase migration is required.
