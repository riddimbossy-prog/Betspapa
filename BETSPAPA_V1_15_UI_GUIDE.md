# BetsPapa v1.15 — Modern Unified UI

## Direction

A new midnight-royal interface with purple intelligence accents, gold highlights and explicit live/result states. The design keeps BetsPapa free and public and avoids wagering prompts.

## Main changes

- One consistent top navigation across all public pages.
- Five-item mobile bottom navigation: Today, Papa, Boss, Live and Results.
- Homepage reorganised around featured analysis, fixture status, HT/FT intelligence and verified results.
- Engine pages use a compact analysis-mode tab bar.
- Match cards have stronger hierarchy for state, teams, market, strength and score.
- Mobile filters open as a bottom sheet instead of occupying the full page.
- Results tables become readable stacked cards on phones.
- Larger touch targets, clearer focus styles and stronger text contrast.
- Desktop sidebar removed from the normal layout; it remains available as the mobile More drawer.
- Responsible-use strip remains visible across public pages.

## Breakpoints

- Desktop: 1280px and above
- Tablet / foldable: 761–1279px
- Mobile: 760px and below
- Narrow phone: 480px and below

## Functional preservation

The redesign does not change PapaSense, OMNI, live status, fixture loading or automatic settlement. Existing IDs and JavaScript hooks are preserved.
